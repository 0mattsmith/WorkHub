# What's New in WorkHub

## 1.0.6
- New "What's New" window that appears after WorkHub updates, so you can see what changed.
- New "What have I missed?" digest on launch, summarising unread messages and notifications per app.
- Add your own custom icon to any app when the website's favicon isn't good enough.
- Custom window titlebar with proper minimise / maximise / close controls (native traffic lights on macOS); the window now starts maximised. You can switch back to the system frame in Settings → General.
- Snooze desktop notifications for a set time, from Settings or the tray menu.
- Remember passwords: WorkHub can save and auto-fill your site logins, encrypted with your device's keychain.
- Right-click an app's icon to "Freeze icon" so it stops changing.
- Red unread count bubbles now appear on app icons from notifications, not just page titles.
- Fixed the Slack "browser no longer supported" message.
- Updates now install silently and relaunch automatically, with a download progress bar — no setup wizard.

## 1.0.5
- Settings menu reorganised into categories with a side navigation.
- Sticky notes and to-do lists, with an option to include them in workspace export/import.
- Fixed showing/hiding the address bar.

## 1.0.4
- Keep embedded apps signed in more reliably (persistent storage + cookie flushing).
- App icon colour picker and right-click address bar to hide it.
